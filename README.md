# DisplayLinkBrunTool
 Program for burning serial number and MAC address for DisplayLink IC.

## Program

- Notices :Copyright(c) 2023 Leno
- Compiler :Embarcadero RAD Studio 2010 Version 14.0.3615.26342
- OS :Windows 8 ver6.3 Build 9600
